Buttons

Probably the first thing people look at when choosing a UI toolkit. The common button!

Basic Button

To make a button a Blaze button add the .citrusStylesSE-button class to buttons, inputs or links.

Link 
<button type="button" class="citrusStylesSE-button">Button</button>
            <input type="button" class="citrusStylesSE-button" value="Input">
            <a class="citrusStylesSE-button">Link</a>
            <button type="button" class="citrusStylesSE-button" disabled>Button</button>

Button Colors

There are five button color modifiers:

<button type="button" class="citrusStylesSE-button">Button</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-button--brand">Button</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-button--info">Button</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-button--warning">Button</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-button--success">Button</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-button--error">Button</button>

Button Groups

To group related buttons together you can use the .citrusStylesSE-input-group wrapping element. This will bunch the buttons together and remove the spacing between them.

<span class="citrusStylesSE-input-group">
              <button type="button" class="citrusStylesSE-button citrusStylesSE-button--brand">Button</button>
              <button type="button" class="citrusStylesSE-button">Button</button>
            </span>

Adding the .citrusStylesSE-input-group--rounded modifier will make the first and last buttons rounded.

<span class="citrusStylesSE-input-group citrusStylesSE-input-group--rounded">
              <button type="button" class="citrusStylesSE-button citrusStylesSE-button--brand">Button</button>
              <button type="button" class="citrusStylesSE-button">Button</button>
              <button type="button" class="citrusStylesSE-button citrusStylesSE-button--info">Button</button>
            </span>

Button Sizes

Adding a size utility class will change the size of the button.

<button type="button" class="citrusStylesSE-button citrusStylesSE-xsmall">.citrusStylesSE-xsmall</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-small">.citrusStylesSE-small</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-medium">.citrusStylesSE-medium</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-large">.citrusStylesSE-large</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-xlarge">.citrusStylesSE-xlarge</button>
            <button type="button" class="citrusStylesSE-button citrusStylesSE-super">.citrusStylesSE-super</button>

Grid

The grid is the easiest way to produce a fluid responsive layout. Its easy to use and very flexible.

color key:

— grids are pink
— cells are blue
— padding is green

Basic Grid

Use the .citrusStylesSE-grid class to wrap your .citrusStylesSE-grid__cell elements.

Offsets

To "push" a cell horizontally across the grid use the .citrusStylesSE-grid__cell--offset-xx class. Use in-conjunction with widths to perfectly align your cells.

first
second

Wrapping

A grid is essentially a row of cells. However cells can wrap onto the next line within the grid.

Here we have 2 cells, 50% and 66.6666% wide, but the grid is only 100% wide...eek!

first
second 
<div class="citrusStylesSE-grid citrusStylesSE-grid--demo">
  <div class="citrusStylesSE-grid__cell citrusStylesSE-grid__cell--width-50">
    <div class="citrusStylesSE-grid-text">first</div>
  </div>
  <div class="citrusStylesSE-grid__cell citrusStylesSE-grid__cell--width-66">
    <div class="citrusStylesSE-grid-text">second</div>
  </div>
</div>

To solve this we can wrap the cells within the grid by adding the .citrusStylesSE-grid--wrap modifier class.

first
second 
<div class="citrusStylesSE-grid citrusStylesSE-grid--wrap citrusStylesSE-grid--demo">
  <div class="citrusStylesSE-grid__cell citrusStylesSE-grid__cell--width-50">
    <div class="citrusStylesSE-grid-text">first</div>
  </div>
  <div class="citrusStylesSE-grid__cell citrusStylesSE-grid__cell--width-66">
    <div class="citrusStylesSE-grid-text">second</div>
  </div>
</div>

Responsiveness

The grid can be instructed to go full width at certain break points to maximise the screen space and help produce great user experiences on all sorts of devices.

The grid has a fit or full class modifier that is used at different screen sizes.

.citrusStylesSE-grid--{small | medium | large}-{fit | full}

first
second
third 
<div class="citrusStylesSE-grid citrusStylesSE-grid--small-full citrusStylesSE-grid--medium-fit citrusStylesSE-grid--large-full citrusStylesSE-grid--demo">
    <div class="citrusStylesSE-grid__cell">
      <div class="citrusStylesSE-grid-text">first</div>
    </div>
    <div class="citrusStylesSE-grid__cell">
      <div class="citrusStylesSE-grid-text">second</div>
    </div>
    <div class="citrusStylesSE-grid__cell">
      <div class="citrusStylesSE-grid-text">third</div>
    </div>
  </div>

The above example will display the cells full width for small and large size screens. For medium screens the cells will adjust to fit the available space. Try it out, either resize your browser or rotate your phone into landscape.

Responsive Suffixes

As well as grid level responsiveness you also have control over the cells within a grid.

Individual cells can change width at specific break points by using responsive suffixes.

small
medium
large 
<div class="citrusStylesSE-grid citrusStylesSE-grid--wrap citrusStylesSE-grid--demo">
  <div class="citrusStylesSE-grid__cell citrusStylesSE-grid__cell--width-100 citrusStylesSE-grid__cell--width-33@small">
    <div class="citrusStylesSE-grid-text">small</div>
  </div>
  <div class="citrusStylesSE-grid__cell citrusStylesSE-grid__cell--width-100 citrusStylesSE-grid__cell--width-33@medium">
    <div class="citrusStylesSE-grid-text">medium</div>
  </div>
  <div class="citrusStylesSE-grid__cell citrusStylesSE-grid__cell--width-100 citrusStylesSE-grid__cell--width-33@large">
    <div class="citrusStylesSE-grid-text">large</div>
  </div>
</div>

As the viewport increases in size the cells go from 100% wide to 33% wide one after another, rather than all at once.

The suffixes apply to the screen sizes and up, in a mobile first manner. In other words:

Set cell width to 33% when screen width is greater than or equal to xsmall | small | medium | large | xlarge | super

Gutters

Blaze's grid comes with a gutter around each grid cell; some padding either side.

Sometimes you want your cells to be flush with each other. To turn off all gutters use .citrusStylesSE-grid--no-gutter

Fixed widths

Usually cells in a grid will be fluid and responsive. If however you want to set a cell to a fixed width add the .citrusStylesSE-grid__cell--width-fixed modifier and give it a specific width.

first
static width of 250px
third 
<div class="citrusStylesSE-grid citrusStylesSE-grid--demo">
  <div class="citrusStylesSE-grid__cell">
    <div class="citrusStylesSE-grid-text">first</div>
  </div>
  <div class="citrusStylesSE-grid__cell citrusStylesSE-grid__cell--width-fixed" style={{width: '250px'}}>
    <div class="citrusStylesSE-grid-text">static width of 250px</div>
  </div>
  <div class="citrusStylesSE-grid__cell">
    <div class="citrusStylesSE-grid-text">third</div>
  </div>
</div>

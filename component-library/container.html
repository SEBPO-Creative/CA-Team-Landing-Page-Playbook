Containers

Containers provide media query screen width restrictions on your content.

<div class="citrusStylesSE-container citrusStylesSE-container--xsmall"></div>
<div class="citrusStylesSE-container citrusStylesSE-container--small"></div>
<div class="citrusStylesSE-container citrusStylesSE-container--medium"></div>
<div class="citrusStylesSE-container citrusStylesSE-container--large"></div>
<div class="citrusStylesSE-container citrusStylesSE-container--xlarge"></div>
<div class="citrusStylesSE-container citrusStylesSE-container--super"></div>

Container sizes

Containers work off the screen size variables:

$screen-width-xsmall:  20em;
$screen-width-small:   30em;
$screen-width-medium:  48em;
$screen-width-large:   64em;
$screen-width-xlarge:  78em;
$screen-width-super:   116em;

Responsiveness

Each container can also be applied at certain screen sizes. .citrusStylesSE-container--small@large for example will restrict the horizontal width to small when the viewport has a large or more width. Each size container has an xsmall, small, medium, large, xlarge and super responsive suffix.
